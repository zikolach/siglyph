# Asciinema Demo Recordings

This page documents local recording for polished terminal demos. These recordings are publishing artifacts. They are not required for compile, test, formatting, lint, or OpenSpec validation commands.

## Requirements

Required only when recording or replaying casts locally:

- `asciinema`

Not required by this workflow:

- Node.js
- npm
- `expect`
- `agg`
- `svg-term`
- browser automation

## Referenced code fragments

The clips are generated by `asciinemaDemo/src/scalatui/demo/AsciinemaDemo.scala`.

The entry point selects a deterministic scenario:

```scala
@main def asciinemaDemo(scenario: String): Unit =
  AsciinemaScenarios.run(scenario)
```

The agent prompt clip uses staged `Input` and `SelectList` frames:

```scala
agentPromptFrame(
  inputValue = "/help Please review @REA",
  suggestions = attachmentSuggestions(),
  footer = "Attachment suggestions wait until the typed prefix is visible."
)
```

The command palette clip uses a fuzzy `SelectList` and tick-driven `Loader`:

```scala
SelectListOptions(filtering = SelectListFiltering.Fuzzy)
Loader(LoaderOptions(message = "Recording release demo", leadingBlankLine = false))
```

The Unicode clip renders a real `Input` value and records typed input labels:

```scala
Input("hello 世界 👩🏽‍💻 café")
Vector("key=Insert", "key=Character(c) modifiers=ctrl")
```

Run a scenario directly when you only want to preview terminal output:

```bash
SIGLYPH_ASCIINEMA_PAUSE_MS=0 mill asciinemaDemo.run agent-prompt
mill asciinemaDemo.run command-palette
mill asciinemaDemo.run unicode-input
```

`SIGLYPH_ASCIINEMA_PAUSE_MS=0` removes frame pauses for quick local checks. Omit it for recording so the cast uses deliberate README-demo pacing.

## Record casts

Record one scenario:

```bash
scripts/asciinema/record-demo.sh agent-prompt
scripts/asciinema/record-demo.sh command-palette
scripts/asciinema/record-demo.sh unicode-input
```

Record all scenarios:

```bash
scripts/asciinema/record-demo.sh all
```

The script writes generated casts and hidden build-tool logs under `artifacts/asciinema`:

```text
artifacts/asciinema/siglyph-agent-prompt.cast
artifacts/asciinema/siglyph-command-palette.cast
artifacts/asciinema/siglyph-unicode-input.cast
artifacts/asciinema/agent-prompt.stderr.log
artifacts/asciinema/command-palette.stderr.log
artifacts/asciinema/unicode-input.stderr.log
```

The script redirects Mill progress, compiler progress, and resolver output to the `.stderr.log` files so the visible cast contains only the scenario.

## Play local casts

```bash
asciinema play artifacts/asciinema/siglyph-agent-prompt.cast
asciinema play artifacts/asciinema/siglyph-command-palette.cast
asciinema play artifacts/asciinema/siglyph-unicode-input.cast
```

Useful playback options:

```bash
asciinema play -s 2 artifacts/asciinema/siglyph-agent-prompt.cast
asciinema play -i 0.1 artifacts/asciinema/siglyph-agent-prompt.cast
```
